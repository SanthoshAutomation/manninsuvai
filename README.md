# Mannin Suvai - Flutter E-Commerce App

> **From our soil to your soul** | பாரம்பரிய சுவை, பரிசுத்தமான வாழ்க்கை

A Flutter-based e-commerce app for **Mannin Suvai** — a traditional homemade health mix brand from Tamil Nadu, India. Supports Android and Web platforms.

---

## Features

- **Splash Screen** — Animated brand intro with logo and tagline
- **Home** — Hero banner carousel, category grid, featured products, "Why Choose Us" section
- **Products Catalog** — Browse by category with search functionality
- **Product Detail** — Size/variant selector, ingredients list, highlights, add to cart
- **Shopping Cart** — Quantity control, order summary, WhatsApp checkout
- **About** — Brand story, contact details, FSSAI registration, social links

## Tech Stack

- **Flutter** 3.x (Dart)
- **Provider** — state management
- **Google Fonts** — Poppins + Playfair Display
- **url_launcher** — WhatsApp & phone links
- **carousel_slider** — Home banner
- **smooth_page_indicator** — Carousel dots
- **badges** — Cart item count
- **flutter_animate** — Splash animations

## Getting Started

### Prerequisites

- Flutter SDK ≥ 3.0.0
- Android Studio / VS Code
- Android SDK (for Android builds)

### Setup

```bash
cd manninsuvai
flutter pub get
flutter run          # mobile (connected device/emulator)
flutter run -d chrome  # web
```

### Build

```bash
# Android APK
flutter build apk --release

# Android App Bundle
flutter build appbundle --release

# Web
flutter build web --release
```
